## 통계학이란?
불확실성을 다루는 학문
자료를 정리하고 분석하여 유용한 정보를 얻기 위한 언어이자 도구

## 데이터와 변수 이해

### 변수

- 관측 대상의 특성을 숫자나 값으로 표현한 것
- 관찰 대상에 따라 서로 다른 값을 가질 수 있는 속성

### 변수의 구분

#### 형태에 따른 분류

1. 범주형 (Categorical)
    1. 명목 척도
    - 순서, 크기 비교의 의미가 없다
    - 빈도, 최빈값 등
    2. 서열척도
    - 순서는 있지만 각 값 사이의 간격이 일정하지 않음
    - 대소, 순위 비교 가능
    - 중앙값, 순위 비교 등
2. 연속형 (Numerical)
    1. 구간척도
    - 값 사이의 간격은 일정함
    - 절대적인 0이 존재하지 않음
    - 평균, 분산 등
    2. 비율척도
    - 절대적인 0을 가지며 산술 연산이 가능함
    - 순서, 간격, 비율 비교 등 가능

#### 기능에 따른 분류

1. 종속변수 (Dependant Variable, Target, Y)
- 독립변수의 영향을 받는 결과 변수

2. 독립변수 (Independent Variable, Input, X)
- 결과에 영향을 주는 원인 변수

3. 통제변수 (Control Variable)
- 결과에 영향을 줄 수 있지만, 연구에서는 일정하게 유지하거나 함께 고려하는 변수
- 종속변수에 미치는 영향을 정확하게 파악하기 위해 통제되는 변수

> 통계는 데이터의 특성에 맞는 분석 방법을 선택하는 과정

## 기술 통계학 (Descriptive Statistics)

- 수집된 자료들이 가지고 있는 정보를 손쉽게 파악할 수 있도록 표현하는 방법에 다루는 학문분야
- 주어진 데이터 집합을 요약하고 정리하는 데 초점을 맞춤
- 중심 경향과 분포 등을 계산하여 데이터의 요약, 분포, 관계 등을 시각화

기술 통계량은 데이터의 특성을 요약하여 이해하는 가장 기본적인 분석 방법이다.
데이터의 중심, 산포, 분포 형태 등을 살펴봄으로써 데이터의 전반적인 특성을 빠르게 파악할 수 있다.

### 중심화 (Central Tendency)

데이터의 대푯값을 나타내는 통계량
1. 평균 (Mean)
    - 모든 값을 더해 개수로 나눈 값
    - 가장 많이 사용되지만 이상치에 민감
2. 중앙값 (Median)
    - 데이터를 크기순으로 정렬했을 때 가운데 값
    - 이상치의 영향을 거의 받지 않음
3. 최빈값 (Mode)
    - 가장 많이 나타나는 값
    - 범주형 데이터에서도 활용 가능

**시계열 데이터에서의 중심값은?**

이동 평균 (Moving average)을 사용한다.<br/>
변동성이 줄어들고 추세 파악에 유리하지만, 실제 변화보다 늦게 반응하고 적절한 윈도 크기 선택이 중요하다.

### 퍼짐 (Dispersion)

산포, 데이터가 얼마나 흩어져 있는지를 나타내는 통계량

1. 분산 (Variance)
    - 평균으로부터 떨어진 정도의 제곱 평균
    - 데이터의 변동성을 수치화
2. 표준편차 (Standard Deviation)
    - 분산의 제곱근
    - 가장 너리 사용하는 산포 지표
3. 범위 (Range)
    - 최댓값 - 최솟값
    - 이상치의 영향을 적게 받음
4. 사분위수 범위 (IQR)
    - 제3사 분위수(Q3) - 제1사분위 수(Q1)
    - 이상치의 영향을 적게 받음

![boxplot](/assets/img/boxplot.png)
IQR은 Box plot을 활용해서 시각화함<br/>
Lower, upper whisker는 1Q, 3Q의 1.5배로 그리지만, min, max 값과 비교해 대체될 수 있다.  

#### 변이계수(Coefficient of Variation, CV)

$CV = \frac{\sigma}{\mu} \times 100\%$
- 평균이 다른 데이터의 퍼짐(변동성)을 공정하게 비교하는 지표
- 평균이 다른 집단 데이터를 비교 시 평균 대비 얼마나 퍼져있는지를 비교해야 함

### 분포 형태 (Distribution Shape)

#### 왜도(Skewness)
어느 방향으로 치우쳐 있는가?
- 데이터 분포의 비대칭 정로를 나타내는 통계량
- 0에 가까울수록 좌우가 대칭인 분포를 의미

#### 첨도 (Kurtosis)
평균 주변에 얼마나 집중되어 있는가?
- 분포의 뾰족함과 꼬리의 두꺼운 정도를 나타내는 통계량
- 꼬리의 두꺼움을 설명하는 지표로도 활용  
$Kurtosis = \frac{\text{E}[(X-\mu)^4]}{\sigma^4}$  

초과 첨도는 비교의 편의를 위해 정규분포의 첨도를 기준(3)으로 하여 빼서 만든 값  
첨도가 높으면 이상치 발생 가능성이 높음  
첨도가 낮으면 데이터가 비교적 고르게 분포

## 추론 통계 (Inferential Statistics)

- 수집된 자료들을 바탕으로 알아내고자 했던 모집단에 대한 추정, 예측, 가설검정 등 결론을 도출하는 방법을 다루는 학문분야
- 수집된 자료를 통하여 원래 알아보고자 했던 문제에 대한 결론을 도출

### 용어 정리

- 모집단(Population): 확률변수 X가 취할 수 있는 모든 값 혹은 대상으로 삼는 대상의 특성 전체
- 모수(Parameter): 관심의 대상이 되는 특성
- 표본(Sample): 모집단의 부분집합
- 통계량(Statistics): 표본을 대표하는 값
- 추론(Inference): 표본 데이터를 바탕으로 모집단에 대한 특성을 추정하는 과정

**표본을 사용하는 이유**: 물리적, 시간적, 경제적 제약으로 일일이 전체를 조사하기 어렵기 때문이다.  

### 자료
통계학은 표본의 자료를 수집, 정리, 요약하고 나아가 요약된 자료를 바탕으로 그 자료의 모태가 되는 모집단에 대해 추론하는 작업.  
분석 대상과 동떨어진 자료를 분석한다면 통계학은 잘 위장된 거짓말이 될 수 있기 때문에 어떤 자료가 제대로 된 자료인가, 어떤 점을 유의해서 살펴봐야 하는가를 고민해야 함.

#### 자료를 얻는 법
1. 실험(Experiment): 주로 자연과학, 의학 분야로 연구자가 실험 대상을 통제하기 용이
2. 관측(Observation): 주로 인문, 사회과학 분야로 많은 경우 실험이 불가능해 연구자는 관측을 통해 자료를 확보해야 함. 관측 대상을 통제하기 어려움

#### 자료의 종류
1. 횡단면 자료
    - 한 시점에서 여러 개체를 관측한 자료
    - 한 시점에 대한 정보를 제공하며, 여러 개체를 비교하는데 활용
2. 시계열 자료
    - 한 개체를 여러 시점에 걸쳐 관측한 자료
    - 시간의 흐름에 따라 수집된 자료로써 변화 패턴을 추적하거나 예측할 때 활용
3. 패널 자료
    - 횡단면과 시계열의 특성을 결합하여 여러 개체를 여러 시점에 걸쳐 관측한 자료
    - 개체 간 차이와 시간적 변화를 동시에 분석

### 표본 선정

표본 선정은 연구 결과의 정확성, 신뢰성, 일반화 가능성에 직접적인 영향을 주는 과정  

#### 대표성, Representativeness
- 표본은 모집단을 대표할 수 있어야 함. 모집단과 표본이 다르면, 표본을 통해 얻은 결론이 모집단에 그대로 적용되지 않을 수 있음
- 표본의 구성이 모집단과 비슷해야만 일반화 가능한 결과를 도출할 수 있음

> 따라서 표본이 모집단의 대표성을 띨 수 있도록 구성해야 함

#### 편향, Bias
- 특정 집단이 과도하게 포함되거나, 특정 특성이 과소 대표될 경우 결과가 왜곡될 수 있음
- 표본 선정 과정에서 편향이 발생하지 않도록 주의해야 함

- 선택 편향(Selection Bias): 표본을 선정하는 과정에서 특정 집단이 과대 대표되거나 과소 대표되는 경우
- 응답 편향(Response Bias): 설문조사나 실험에서 응답자의 성향이나 선택에 의해 결과가 왜곡되는 경우

심슨의 역설: 여러 그룹의 자료를 합했을 때의 결과와 각 그룹을 구분했을 때의 결과가 다른 상황. 표본 선정 과정뿐 아니라 변수 간 관계 분석 등에서도 유의해야 함

### 확률 변수
사전적 분포(Distribution) 하에, 사후적으로 데이터를 집어넣으면 Distribution 상에서 하나의 숫자로 확정이 되는 값  
**왜 필요할까?**  
- 표본 공간에서 어떤 사건의 확률을 구하는 것이 매우 복잡하거나 어려울 때가 있음
- 이때 표본공간에서 사건을 실제 값으로 사상시키고, 사건의 확률을 실제 값 위에서 정의된 분포함수를 통해서 구할 수 있음
- 확률 변수는 현실 문제에 확률론을 적용하여 불확실성을 수치로 모델링 할 수 있게 함

#### 이산 확률 변수
셀 수 있는(Countable) 값에 대한 확률
- 표본공간에서 정의된 실수함수, 즉 앞면의 개수 X를 확률변수라 할 수 있음
- 확률변수는 흔히 대문자로 나타내며, 확률변수가 취하는 값들은 소문자로 나타냄
- $f(x)$는 확률변수 $X$의 확률을 나타내는 함수

확률질량함수(PMF; Probability Mass Function): 이산확률변수 $X$가 $x$의 값을 취할 확률을 나타내는 함수  
$f(x) = \text{P[X=x]}$

#### 연속 확률 변수
- 확률 변수X가 취할 수 있는 값들이 무한개인 경우
- 값이 연속적인 구간(실수 범위)에서 나올 수 있는 확률 변수

확률 밀도 함수(PDF; Probability Density Function)
- 특정 값 하나가 나올 확률은 0
- 구간 단위로 확률을 계산 -> 적분  
$P(a \le X \le b) = \int_{a}^{b} f(x)\, dx$  

### 데이터 분포 요약

#### 도수분포표(Frequency table)
- 자료의 분포를 몇 개의 구간으로 나누고, 나누어진 각 구간에 속하는 자료가 몇 개인지 정리한 표
- 구간을 나누면 개별적인 데이터를 보는 것 보다 데이터의 전체적인 분포를 요약해서 볼 수 있다

#### 히스토그램(Histogram)
- 도수분포표를 시각적으로 표현한 막대 그래프
- 데이터가 어떤 범위에 집중되어 있는지, 어떤 구간에 값들이 분포하는지를 쉽게 파악할 수 있음
- X축은 데이터 값이나 구간, Y축은 빈도(해당 구간에 포함된 데이터의 개수)나 비율로 나타낼 수 있음

### 정규분포, Normal Distribution

이상적인 히스토그램, 하나의 수학적 모형, 개념상 모집단의 분포

#### 정의
- 통계학에서 가장 널리 쓰이는 확률분포 중 하나로 데이터가 평균을 중심으로 좌우 대칭으로 분포하는 확률 분포
- 가우시안 분포(Gaussian  distribution)이라고도 함

$f(x) = \frac{1}{\sigma \sqrt{2 \pi}}e^{-\frac{(x-\mu)^2}{2\sigma ^2}}$  
- $\mu$: 모집단의 평균
- $\sigma$: 모집단의 표준편차
- $\sigma ^2$: 분산, 표준편차의 제곱
- $e$: 자연상수  
$\text{N}(\mu, \sigma^2)$

#### 특징
- 봉우리가 하나
- 종 모양의 곡선
- 평균을 중심으로 좌우 대칭
- 곡선 아래 면적(AUC)가 1

### 표준정규분포, Standard Normal Distribution
확률 계산을 용이하게 하고, 다른 정규 분포간 비교 가능성을 확보

#### 정의
- 평균이 0이고, 표준편차가 1인 정규분포
- 다른 정규분포의 값을 표준화하여, 해당 값의 확률을 쉽게 구할 수 있음
- 통계정 추정 및 가설 검정에서 중요한 역할을 함

#### Z-Score 변환
- 표준화된 값(z-score)를 통해 다른 정규분포를 변환
- 각 관측치에서 평균을 빼고 편차로 나누어 줌

$z=\frac{x-\mu}{\sigma}$  
- $x$: 모집단의 원본 값
- $\mu$: 모집단의 평균
- $\sigma$: 모집단의 표준편차

#### 68-95-99.7의 법칙
정규분포下 손쉬운 확률값 도출  

**의미**
- 68%의 데이터는 평균 $\pm 1$ 표준편차 범위 내에 위치
- 95%의 데이터는 평균 $\pm 2$ 표준편차 범위 내에 위치
- 99.7%의 데이터는 평균 $\pm 3$ 표준편차 범위 내에 위치

### 기대값, Expected Value
확률분포에서 평균을 나타내는 개념

#### 정의
- 확률변수가 가질 수 있는 평균적 값을 의미
- 하나의 확률 과정에 의해 결정되는 숫자는 하나의 값 주위로 분포
- 확률 변수의 각 가능한 값에 그 값이 나올 확률을 곱한 후, 그 값을 모두 더한 것

### 표준 오차, Standard Error
불확실성을 측정

#### 정의
- 실현된 값이 기댓값과 차이가 나는 정도
- 하나의 확률 과정에 의해 결정되는 숫자는 기댓값 주위로 분포하며, 그 값은 기댓값 + 표준오차의 차이를 가짐
- 표존 평균이 모집단 평균을 얼마나 정확하게 추정하는지를 나타내는 지표

### 제곱근의 법칙, Square Root Law
#### 정의
- 표준오차는 여러 표본들로부터 얻어낸 표본통계량의 변량으로 표본 평균이 모집단 평균을 얼마나 잘 추정하는지를 나타냄
- 표본의 합의 표준오차는 샘플 수의 제곱근 $\sqrt{n}$에 비례해서 늘어남

### 중심극한정리, Central Limit Theorem
표본의 크기가 커질수록, 표본 평균의 분포는 정규분포에 가까워짐

## 가설 검정

### 가설 검정의 원리
#### 증명의 종류
1. 연역법: 참인 명제를 바탕으로 또 다른 참인 명제를 이끌어내는 방식
2. 귀납법: 제한적 관찰을 통해 공통의 명제를 끌어내거나 법칙을 형식화 하는 방식
3. 귀류법: 어떤 명제의 부정의 모순을 이끌어 냄으로써 그 명제가 참임을 증명하는 방식  

"통계학에서는 검정하고자 하는 어떤 명제가 있을 때 모순에 의한 논증법을 사용함"

### 유의성 검정 프로세스

#### 1. 가설 설정

귀무가설
- 관측된 차이는 우연이라는 주장
- 그 발견이 단지 우연일 뿐이라는 주장  

대립가설
- 관측된 차이가 실질적이라는 주장  

**증명하고자 하는 명제를 대립가설로 설정하고, 이를 부정하는 명제를 귀무가설로 설정함**

#### 2. 데이터로부터 통계량 계산
- 주어진 자료로부터 가설 검정에 사용할 통계량을 계산
- 자료에서 얻은 통계량과 귀무가설 하에서 기대되는 값 사이의 차이를 측정
- 차이를 표준오차로 나누어 표준단위로 나타내면 검정통계량

$z = \frac{x-\mu}{\sigma}$

#### 3. P-value 산출
- 검정통계량은 관측치와 귀무가설 하의 기댓값의 표준화된 차이를 의미하는데, 이로부터 확률을 얻을 수 있음
- 이렇게 구해진 확률을 **P-value**라고 부름  
P-value의 의미: 귀무가설 하의 표본통계량이 관측치보다 극단적인 값으로 얻어질 확률  
P-value가 매우 작다면(일반적으로 0.05) 귀무가설을 기각할 수 있다. 

#### 4. 유의수준 설정
P-value 파악의 척도. 유의수준을 기준으로 하여 P-value의 크고 작음을 가늠한다.  
- $\alpha$로 표기, 검정의 크기라고도 한다.
- 설정한 유의수준의 클수록 귀무가설을 기각할 가능성이 커진다.
- 설정한 유의수준이 낮을수록 엄격한 연구

<u>보편적으로 사용되는 유의수준</u>

- 10% 유의수준 ($\alpha$=0.1), 90% 신뢰도, z값은 1.645
- 5% 유의수준 ($\alpha$=0.05), 95% 신뢰도, z값은 1.96
- 10% 유의수준 ($\alpha$=0.01), 99% 신뢰도, z값은 2.575

#### 5. 판정
- P-value가 유의수준보다 작거나 같으면 귀무가설을 기각
- 귀무가설을 기각하는 경우, 결과가 통계적으로 유의미하다 라고 한다.

**단측검정 vs 양측검정**

| 구분 | 단측검정 | 양측검정 |
| -- | -- | -- |
| 목적 | 특정 방향의 차이를 검정 | 차이의 존재 여부를 검정 |
| 대립가설($\text{H}_1$) | $\mu > \mu_0$ 또는 $\mu < \mu_0$ | $\mu \ne \mu_0$ |
| 기각역 | 한쪽 꼬리 | 양쪽 꼬리 |
| 활용 | 방향성이 명확한 연구 | 대부분의 일반적인 연구 |

유의사항
- 가설검정은 "실질적인 것인가? 단순히 우연에 의한 것인가?"를 판단할 뿐, 차이의 의미와 원인을 알려주지 않음
- 가설검정의 성격은 확률적이지 단정적이지 않음

### t-분포의 의미
불확실성이 큰 작은 표본을 위한 보수적인 추정

#### t-분포란?
- 표본의 크기가 작을 때(특히 $n<30$) 모평균을 추정하기 위해 표본평균의 분포를 정규분포 대신 근사하는 분포
- 표본 수가 커지면 $(n \rightarrow \infty)$ 정규분포와 동일해짐

### 변수 형태에 따른 검정 방식
가설 검정 프로세스는 동일하지만, 사용하는 검정분포와 검정 통계량은 달라진다.

|기법|변수A|변수B|검정 분포|검정 통계량|적용 예|
|-|-|-|-|-|-|-|
|카이제곱 검정|이산형|이산형|카이제곱분포|카이제곱통계량|학력과 결혼 사이에 유의한 관계가 있는가?|
|독립표본 t검정|이산형<br/>(2그룹/독립)|연속형|t-분포|t-통계량|성별에 따른 결혼비율의 차이가 있는가?|
|대응표본 t검정|이산형<br/>(2그룹/Pair)|연속형|t-분포|T-통계량|특별과외 후 A반의 영어성적이 상승했는가?|
|일월 분산분석<br/>(ANOVA)|이산형<br/>(2그룹 이상)|연속형|F-분포|F-통계량|거주지역에 따른 평균 자산액의 차이는 있는가?|

- 가설검정 프로세스는 동일하지만, 데이터의 형태에 따라 적절한 검정 도구를 선택하는 것이 필요
- 달라지는 것은 '데이터의 형태'이며, 데이터의 형태에 맞는 검정통계량($\text{t}, \chi ^2, \text{F}$)을 선택하는 것이 필요  

#### t-검정
- 평균이 얼마나 차이나는가?
- 평균 차이(t)를 관측

$\frac{\text{평균차이}}{\text{평균의 표준오차}}$

#### 카이제곱 검정
- 예상보다 얼마나 많이(적게) 발생했는가?
- 빈도 차이 $(\chi ^2)$ 를 관측

$\frac{\text{관측빈도}-\text{기대빈도}}{\text{기대빈도}}$

#### ANOVA
- 그룹 차이가 우연이라고 보기 어려운가?
- 집단 간 차이와 집단 내 차이를 비교 → 분산차이(F)

$\frac{\text{집단 간 분산}}{\text{집단 내 분산}}$
